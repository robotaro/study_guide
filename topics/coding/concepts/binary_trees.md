- **Balanced Binary Tree**: Height is kept minimal (e.g., AVL Tree, Red-Black Tree).
- **Binary Search Tree (BST)**: The left subtree contains smaller values, and the right subtree contains larger values.
	- Red-black tree is a type of self-balancing Binary Tree
